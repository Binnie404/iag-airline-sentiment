# IAG Airline Sentiment Dashboard

Interactive passenger review analysis covering Aer Lingus, British Airways,
and Iberia (2014–2026) — built as a single self-contained HTML file.

**Live:** https://[yourusername].github.io/iag-airline-sentiment

### Views
- Overview — ratings, sub-metrics, trend & distribution
- Aircraft — performance by aircraft family
- Routes — world arc map with per-route rating breakdown  
- Key Findings — visual report of the major data stories

### Data
3,942 reviews sourced from Skytrax. All data is embedded directly
in the HTML — no server, API, or external files required.

### Stack
Chart.js · D3.js · TopoJSON · Syne
